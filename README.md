# description

gin web framework with mongo db database

# installation

- `go get -u github.com/gin-gonic/gin`
- `go get go.mongodb.org/mongo-driver`
- `go get -u github.com/stretchr/testify/assert`

# run

- `go run main.go`

#  compile into binaries

- `go build main.go`
- `main.exe`

# run test

- `go test -v ./test`
